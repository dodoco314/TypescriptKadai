# Premium TypeScript Calculator (TS Calc)

A sleek, responsive, and feature-rich calculator web application built with **TypeScript** and **Vite**. Features a modern UI with glassmorphism, fluid animations, theme support, scientific operations, and history logging.

## ✨ Features

- **Modern Glassmorphic UI**: Vibrant, responsive design with smooth animations.
- **Dual Themes**: Easily switch between Dark Mode and Light Mode.
- **Scientific Calculator**: Expandable panel for scientific calculations:
  - Trigonometric functions (sin, cos, tan) with Degree/Radian toggle.
  - Constants ($\pi$, $e$).
  - Power & Root functions ($x^y$, $x^2$, $x^3$, $\sqrt{x}$, $\sqrt[3]{x}$).
  - Logarithms ($\log$, $\ln$).
- **Calculation History**: View and manage previous calculations.
- **Keyboard Support**: Full keyboard mapping for standard calculator inputs.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm (or yarn / pnpm)

### Installation & Run

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/dodoco314/TypescriptKadai.git
   cd TypescriptKadai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open the local server URL (usually `http://localhost:5173`) in your browser.

4. **Build for production**:
   ```bash
   npm run build
   ```
